# TechStore - Интернет-магазин техники

Статический веб-сайт интернет-магазина техники, разработанный в соответствии с требованиями ГОСТ Р 56939-2016.

## Структура проекта

## Страница 404

Проект включает кастомную страницу 404 (`404.html`) с следующими функциями:

### Особенности:
- **Семантическая разметка** - соответствует требованиям доступности
- **Адаптивный дизайн** - корректно отображается на всех устройствах
- **Полезные действия** - кнопки для возврата на главную, в каталог или назад
- **Контактная информация** - данные службы поддержки
- **Интеграция с поиском** - поисковые запросы переносятся на главную страницу
- **Соответствие ГОСТ** - включает все необходимые мета-теги и структуру

### Функциональность:
- Автоматическое обновление счетчика корзины
- Перенос поисковых запросов на главную страницу
- Логирование для аналитики
- Навигация по сайту

### Настройка сервера:
Для корректной работы страницы 404 настройте сервер для перенаправления:
- **Apache**: Добавьте в `.htaccess`: `ErrorDocument 404 /404.html`
- **Nginx**: Добавьте в конфиг: `error_page 404 /404.html;`
- **IIS**: Настройте через панель управления или web.config
